## 06 - Tender Folders structure (Template)

### Main Content
This folder contains a template directory structure for organizing tender/project documents. It provides a standardized folder hierarchy for managing project documentation throughout the tender and project lifecycle.

### Subfolders Structure

1. **01 Offer/** - Documents related to offers/quotations
   - **01 In Progress/** - Current/active offer versions
   - **02 Previous Issues/** - Previous versions of offers

2. **02 Estmt/** - Estimation documents
   - **01 In Progress/** - Current estimation work
   - **02 Previous Issues/** - Previous estimation versions

3. **03 Rcvd/** - Received documents
   - **01 From Client/** - Documents received from clients
   - **02 From Sub vendor/** - Documents received from subcontractors/vendors

4. **04 Issd/** - Issued documents
   - **01 To Client/** - Documents sent to clients
   - **02 To Subvendors/** - Documents sent to subcontractors/vendors

5. **05 Eng/** - Engineering documents
   - **01 In Progress/** - Current engineering work
   - **02 Previous Revs/** - Previous engineering revisions

6. **06 RFQ references/** - Request for Quotation reference documents

7. **07 Outcome/** - Final project outcomes
   - **01 Commercial/** - Commercial outcomes
     - **01 In Progress/** - Current commercial deliverables
     - **02 Previous Issues/** - Previous commercial versions
   - **02 Technical/** - Technical outcomes
     - **01 In Progress/** - Current technical deliverables
     - **02 Previous Issues/** - Previous technical versions

### Files
- All subfolders appear to be empty (template structure only)

### Purpose
Provides a standardized organizational structure for managing tender/project documentation. This template ensures consistent document organization across projects, making it easier to:
- Track document versions (In Progress vs. Previous Issues/Revs)
- Organize documents by type (Offers, Estimates, Engineering)
- Separate incoming vs. outgoing documents (Received vs. Issued)
- Maintain clear separation between commercial and technical outcomes
- Archive RFQ reference materials

This structure supports efficient document management and version control throughout the project lifecycle.
